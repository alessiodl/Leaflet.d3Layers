Leaflet.d3Layers

Leaflet.d3Layers is a collection of JavaScript functions used to easily draw D3.js layers on top of a map built with the Leaflet web mapping library. The principal advantage in using D3 is the possibility to manage large amount of spatial data as SVG object, mantaining very high performances: no more browser freeze or unresponsive maps!

The plugin is under active development and it is really not recommended for your production environment!
